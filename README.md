## 🖼️ القوالب المتاحة

### 1️⃣ عصري (Modern)
> تصميم عصري بشريط جانبي ملوّن يحتوي على الصورة والمعلومات الشخصية، مع المحتوى الرئيسي على الجانب الآخر.

### 2️⃣ كلاسيكي (Classic)
> تصميم تقليدي مع رأس صفحة مركزي يتضمن الاسم والمسمى الوظيفي، مثالي للوظائف الرسمية والأكاديمية.

### 3️⃣ إبداعي (Creative)
> تصميم جريء مع شريط رأس عريض ملوّن بالكامل، مثالي لمجالات التصميم والإبداع.

## 🚀 التشغيل

هذا المشروع **لا يحتاج أي تثبيت** - فهو يعمل مباشرة في المتصفح!

### الطريقة الأسهل:
```
افتح ملف index.html في أي متصفح حديث
```

### باستخدام Live Server (مستحسن):
```bash

# ثبّت إضافة Live Server ثم اضغط "Go Live"

# أو باستخدام npx
npx live-server .
```

> **📌 ملاحظة:** المشروع يعمل بشكل كامل بدون خادم (server) عبر بروتوكول `file://`.

## 📖 طريقة الاستخدام

### 1️⃣ ملء البيانات
انتقل بين التبويبات لملء بياناتك:

| التبويب | المحتوى |
|---------|---------|
| 👤 **شخصي** | الاسم، المسمى الوظيفي، النبذة الشخصية |
| 📞 **التواصل** | البريد، الهاتف، الدولة |
| 💼 **الخبرات** | الخبرة العملية (إضافة عدد غير محدود) |
| 🎓 **التعليم** | الشهادات الأكاديمية (إضافة عدد غير محدود) |
| 🛠️ **المهارات** | المهارات مع مستوى الإتقان (شريط تقدم) |
| 📋 **إضافي** | الحالة الاجتماعية، الخدمة العسكرية، اللغات |

### 2️⃣ اختيار القالب واللون
- اختر القالب من القائمة المنسدلة أعلى المعاينة
- اختر اللون من دوائر الألوان أو استخدم Color Picker لاختيار لون مخصص

### 3️⃣ تخصيص إضافي
- 📸 **ارفع صورتك** عبر زر "تحميل صورة"
- ✍️ **ارسم على الـ CV** عبر زر "رسم" لإضافة ملاحظات أو توقيع

### 4️⃣ التصدير
- 📄 **PDF** - تصدير كملف PDF جاهز للطباعة
- 🖼️ **JPG** - تصدير كصورة عالية الجودة

## 🧩 البنية البرمجية

المشروع مبني بشكل **معياري (Modular)** داخل ملف واحد لتجنب مشاكل CORS:

```
main.js
│
├── 📦 storage        # حفظ/تحميل البيانات من LocalStorage
├── 🎨 themes         # إدارة الثيمات والألوان
├── 📐 templates      # قوالب HTML للسيرة الذاتية (Modern, Classic, Creative)
├── 📝 form           # إدارة النماذج والتبويبات والإدخال
├── ✏️ drawing        # أدوات الرسم على Canvas
├── 📸 photo          # رفع ومعالجة الصور
├── 📤 exporter       # التصدير إلى PDF و JPG
├── 📄 paginator      # تقسيم المحتوى على صفحات A4 تلقائياً
└── 🎮 App            # المتحكم الرئيسي - يربط كل الوحدات
```

### خوارزمية تقسيم الصفحات (Paginator):
يستخدم المشروع نظام تقسيم ذكي:
1. يجمع كل عناصر الـ CV (أقسام، خبرات، مهارات...)
2. يضيف العناصر واحداً تلو الآخر في صفحة وهمية للقياس
3. إذا تجاوز المحتوى ارتفاع صفحة A4 (`1115px`)، ينشئ صفحة جديدة
4. يكرر العملية حتى توزيع كل المحتوى

## 📱 الاستجابة للأجهزة

| الجهاز | العرض | السلوك |
|--------|-------|--------|
| 🖥️ **كمبيوتر** | `> 768px` | عرض المحرر والمعاينة جنباً إلى جنب |
| 📱 **موبايل** | `≤ 768px` | شريط تنقل سفلي للتبديل بين المحرر والمعاينة |

### التصغير الذكي على الموبايل:
يتم تصغير صفحة الـ A4 (`210mm × 297mm`) تلقائياً لتناسب عرض شاشة الموبايل باستخدام CSS Transforms مع الحفاظ على جودة المحتوى.

## 🔧 التخصيص

### إضافة ثيم جديد:

**1. في `css/variables.css`:**
```css
[data-theme="my-theme"] {
    --primary: #FF5722;
    --primary-dark: #E64A19;
    --primary-light: #FFF3E0;
}
```

**2. في `js/main.js` (مصفوفة الثيمات):**
```javascript
{ id: 'my-theme', color: '#FF5722' }
```

### إضافة قالب جديد:

1. أضف دالة القالب في كائن `templates` في `main.js`
2. أضف تنسيقات القالب في `cv-preview.css`
3. أضف خيار القالب في القائمة المنسدلة في `index.html`
